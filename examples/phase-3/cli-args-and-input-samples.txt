--- resolveTreeText(sources) ---

  => ".\n└── ++ a.ts"

resolveTreeText({ text: 'a', file: null, stdin: 'b' })     # two modes
  => throws CliError: exactly one Change Tree input must be provided

resolveTreeText({ text: null, file: null, stdin: null })   # no mode
  => throws CliError: Change Tree input is required

resolveTreeText({ text: '   ', file: null, stdin: null })  # whitespace-only
  => throws CliError: Change Tree input is required

resolveTreeText({ text: null, file: 'missing.txt', stdin: null })
  => throws CliError: Change Tree file could not be read: missing.txt

--- USAGE ---

Usage: change-tree-svg [options]

Input (exactly one):
  -t, --text <tree>   Change Tree notation as a string
  -f, --file <path>   read Change Tree notation from a file
  (or pipe Change Tree notation on stdin)

Output:
  -o, --output <path> write SVG to a file instead of stdout
  --no-fallback       omit the plain-text fallback

Other:
  --no-legend         hide the legend in SVG and fallback
  -h, --help          show this help
